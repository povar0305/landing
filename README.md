# Vape-Zona landing

## Верстка лендинга для компании Vape Zona.
Исходник - сайт конкурентов
Верстка мобильной версии - по правкам клиента

Использовано:
* Bootstrap 5v
* HTML 5
* Sass
* Animate.css + WOW js - для анимации 
* БЭМ для нейминга
* OptimizedHTML 5
